🚀 COMO USAR - PASSO A PASSO:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1️⃣  EXTRAIR O ZIP
      ├─ Clique direito no arquivo: dist\WA-Multi-1.3.0-Portable.zip
      ├─ Selecione: "Extrair tudo..." ou "Extract here"
      └─ Escolha uma pasta (ex: C:\Programas\WA-Multi)


  2️⃣  EXECUTAR O PROGRAMA
      ├─ Abra a pasta extraída
      ├─ Duplo-clique em: "WA Multi.exe"
      └─ O aplicativo abrirá automaticamente


  3️⃣  FAZER LOGIN (PRIMEIRA VEZ)
      ├─ Você verá 2 sessões: "WA A" e "WA B"
      ├─ Clique em uma sessão
      ├─ Escaneie o QR Code com seu WhatsApp
      └─ Repita para outras sessões


  4️⃣  ADICIONAR MAIS SESSÕES
      ├─ Clique no botão: "+ Adicionar"
      ├─ Uma nova sessão será criada (C, D, E...)
      └─ Faça login escaneando o QR Code


  5️⃣  USAR OS CONTROLES
      ├─ [Remover]    → Apaga a sessão selecionada
      ├─ [Maximizar]  → Expande a sessão para tela cheia
      ├─ [Restaurar]  → Volta ao modo grade
      ├─ [−10%]       → Diminui o zoom
      ├─ [+10%]       → Aumenta o zoom
      └─ [Recarregar] → Atualiza a página


💾 ONDE FICAM OS DADOS?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  📁 [Pasta onde você extraiu]\data\

  Exemplo:
  C:\Programas\WA-Multi\
  ├── WA Multi.exe
  ├── LEIA-ME.txt
  └── data\                    ← Seus dados ficam aqui
      ├── state.json           ← Configurações
      └── Partitions\          ← Logins salvos
          ├── persist_wa_A\
          ├── persist_wa_B\
          └── ...


⚠️  IMPORTANTE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ❌ NÃO delete a pasta "data" se quiser manter os logins salvos!
  
  ✅ Para mover para outro computador:
     → Copie TODA a pasta incluindo "data"
     
  ✅ Para começar do zero:
     → Feche o app
     → Delete a pasta "data"
     → Reabra o app

❓ PROBLEMAS COMUNS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ❌ QR Code não aparece
     ✅ Solução: Clique em "Recarregar"

  ❌ Sessão pede QR code novamente
     ✅ Solução: Normal, depende do WhatsApp. Basta escanear.

  ❌ Tela em branco
     ✅ Solução: Feche e reabra o app

  ❌ WhatsApp pede para atualizar navegador
     ✅ Solução: Já está configurado com Chrome moderno, ignore
